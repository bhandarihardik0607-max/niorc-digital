4. Database Schema
Table	Purpose
profiles	Vendor business info (business_name, owner_name, phone, google_review_link, onboarding_status, preferred_language)
customers	Customer data (real_name, phone, gender, visit_count, total_spend, favorite_item, opted_out, engagement_status)
bills	Transaction records (customer_id, vendor_id, items, total_amount, status, whatsapp_message_id)
menu_items	Menu catalog (name, price, size, category, is_available)
inventory	Stock tracking (item_name, current_stock, min_stock_level, unit, menu_item_id)
loyalty_points	Points system (customer_id, points, lifetime_points, tier: bronze/silver/gold/platinum)
loyalty_rewards	Reward definitions (name, points_required, reward_value, reward_type)
reward_redemptions	Redemption history
notifications	In-app notifications (title, message, type, is_read, link)
scheduled_messages	Queued WhatsApp messages
user_roles	Admin/user role management
5. Edge Functions
Function	Purpose
send-whatsapp-bill	Send bill summary to customer via WhatsApp Business API
whatsapp-webhook	Handle incoming WhatsApp messages (opt-out: "STOP", "unsubscribe", "रुको")
generate-automations	Create automated follow-up messages for inactive customers
extract-menu-items	AI-powered menu extraction from uploaded images (Gemini)
create-cashfree-order	Create Cashfree payment orders for subscriptions
cashfree-webhook	Handle Cashfree payment callbacks (HMAC verified)
send-welcome-email	Send welcome email via Resend API

9. WhatsApp Integration
Required Secrets:

WHATSAPP_ACCESS_TOKEN
WHATSAPP_PHONE_NUMBER_ID
WHATSAPP_VERIFY_TOKEN (for webhook)
Communication Rules:

Default language: Hinglish (warm & respectful)
Anti-spam: Max 2-3 messages per customer per week
Every message includes: "Reply STOP to opt out"
Opted-out customers are NEVER messaged again
Opt-Out Keywords: stop, unsubscribe, opt out, रुको, बंद करो, रोको
